# A Simple GPT

## 项目简介

这是一个简单的GPT（生成式预训练转换器）项目，旨在提供一个用于初学者快速掌握GPT。

## 安装说明

### 环境要求

- Python 3.8+
- PyTorch 1.9+
- CUDA（可选，用于GPU加速）

### 安装步骤

1. 克隆仓库
```bash
git clone https://github.com/lyramilk/a_simple_gpt.git
```

2. 安装依赖
```bash
pip install -r requirements.txt
```

## 运行

```bash
cd a_simple_gpt/gpt
python gpt0.py
```
